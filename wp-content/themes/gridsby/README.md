Gridsby

===

a WordPress theme for selling your photos and digital prints.

Gridsby is a pinterest style gallery theme which is a beautiful, simple to showcase a beautiful photo collection. Gridsby is responsive and retina ready, and includes an easy method for posting photos to a front page gallery.
